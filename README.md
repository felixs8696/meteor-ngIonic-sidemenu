# Starter Template for a Meteor-Angular-Ionic Project
A Meteor Project with AngularJS, MongoDB, and Ionic Sidemenu template set up. Javascript is the ECMAScript 2015 (ES6) version.
Built using the `meteor CLI`

### Platforms
* `ios`
* `android`
